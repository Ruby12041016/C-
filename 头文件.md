## 记住的要点

### 容器选择指南：
| 需求 | 推荐容器 | 原因 |
|------|----------|------|
| 快速随机访问 | vector | O(1)访问 |
| 快速查找（按键） | unordered_map | O(1)平均查找 |
| 有序遍历 | map | 自动排序 |
| 去重+排序 | set | 自动去重排序 |
| 频繁头尾操作 | deque | O(1)头尾操作 |

### 算法使用模式：
```cpp
// 1. 排序
std::sort(vec.begin(), vec.end());

// 2. 查找
auto it = std::find(vec.begin(), vec.end(), value);

// 3. 计数
int count = std::count(vec.begin(), vec.end(), value);

// 4. 删除特定元素
vec.erase(std::remove(vec.begin(), vec.end(), value), vec.end());

// 5. 遍历并操作
std::transform(vec.begin(), vec.end(), result.begin(),
               int x { return x * 2; });
```

---
1. **优先使用标准库**，不要自己造轮子
2. **容器选型**：80%的情况用vector，15%用map/unordered_map，5%用其他
3. **智能指针**：优先使用unique_ptr，需要共享时用shared_ptr
4. **Lambda**：简短的操作优先用lambda
5. **范围for循环**：优先于传统for循环
6. **auto关键字**：让编译器推导类型
7. **异常安全**：使用RAII（资源获取即初始化）
